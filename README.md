Database-Project
This repository contains final Project of Database Systems course studied regular in Spring 2023 BS Data Science, University of the Punjab, Lahore
Project is deployed and fully functional online and can be visited at https://hamza-safwan.github.io/ or http://dbproject.pythonanywhere.com/

How to run the project?
1. Install python 3.6 or above.
2. Install flask 1.1.2 or above
3. Run this command to get required dependencies pip install -r requirements.txt
4. Finally, run this command flask run
5. If everything went correctly,then project will be available at http://127.0.0.1:5000/ or http://localhost:5000/.

Technologies used
Front-end:- HTML,CSS,Bootstrap,JavaScript
Back-end:- Python
DataBase:- MySQL
